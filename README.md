This project analyzes political crisis and insurgencies in Nigeria using structured survey data and Excel-based data analytics tools.
The dashboard transforms raw data into meaningful insights that help identify:
Most affected regions
Frequency of crisis incidents
Impact distribution across demographics
Trends over time
Public perception and responses
The goal is to turn complex socio-political data into a clear, interactive visual story.

Project Overview
This project was built using primary data collected through a self-designed questionnaire, demonstrating end-to-end research and analytics capability from survey design to insight visualization.

The dashboard transforms raw survey data into structured, interactive visual insights to support data-driven understanding of socio-political instability.
